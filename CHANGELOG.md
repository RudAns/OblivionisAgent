# 更新日志 / Changelog

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。日期格式 YYYY-MM-DD。

## [0.1.0] - 2026-06-16

首个公开版本。把飞书群聊接入本地 Claude Code 会话的 Windows 桌面工具。

### 🧩 核心
- **飞书群 ↔ 本地 Claude 会话桥接**：群里 @机器人 → 路由到你电脑上对应项目的 claude 会话 → 富文本回帖并 @提问人。
- **订阅合规**：一切 LLM 调用都通过遥控**官方 `claude` CLI** 完成，绝不直连 API / 碰订阅 OAuth 令牌。
- **连线式画布**（React Flow）：飞书群 → 路由 → Claude 会话，拖节点连线即接入；可折叠成会话卡片专注终端。
- **两会话模型**：`base`（你的开发终端会话，永不被飞书污染）+ `fork`（脱敏分身，所有飞书消息都走它）。
- **多群多会话 + 意图分流**：不同群路由到不同项目（各自 cwd / 模型 / 权限）；同群消息按 LLM 语义意图走不同分支。

### 🔗 节点类型
- 飞书群 / 路由 / 意图分流 / Claude 会话 / 定时任务 / Webhook。
- **赋能节点三件套**：🎭 人格（SOUL.md）/ 🧩 技能（SKILL.md）/ 🦾 子代理（Claude Code 原生 subagent，在独立上下文做重活、claude 自动委派）——连到会话的赋能口即生效，一个节点可共享给多个会话。
- 赋能连线用**虚线 + 按类型上色**与消息流区分。
- 「**重锚人格**」：改完人格 / 刚连上会话后一键把当前人格重新注入所连会话（保留对话记忆），比「刷新快照」更轻。

### 🔐 安全
- **访客隔离**：访客走 fork 脱敏分身；fork 时 transcript 抹密钥；回复**出站二次脱敏** + 访客安全护栏。
- **飞书审批卡**：访客的敏感操作（改文件 / 执行命令）弹交互卡到群里，主人 [允许 / 拒绝] 才执行；裁决或超时后卡片自动更新状态、去按钮。
- **App Secret 存 Windows 凭据管理器**：不再明文落 `config.json`、不经 WS 广播；首次启动自动从旧明文迁移。
- 主人 / 访客分级 permission mode；全量入站审计落盘。

### 💬 飞书侧交互
- 自助命令：`/status`（状态卡）· `/doctor`（自检）· `/retry`（重跑）· `/continue`（续跑），仅主人可用。
- 读飞书 **docx / Wiki / Sheets / 多维表** 喂上下文。
- **长回复自动作为飞书文件回传**，不撑爆气泡。
- 回复含 Markdown 表格 → 飞书**原生表格**。
- **逐工具进度**流式卡：实时显示「正在 运行命令 / 读取文件…」。

### 🖥️ 桌面体验
- **内置交互终端**：双击节点打开开发会话（完整历史回放）；多终端保活·拖拽排序、剪贴板贴图喂图、字号缩放（设置滑杆 / Ctrl±）。
- **中英双语界面**：设置里一键切换 中文 / English，技术标识符保持原样，漏译自动回退中文。
- **会话视图粘滞**：左侧选会话只换「哪个会话」，终端 / 转录视图用面板顶部小页签切换，不再强制跳终端。
- **定时任务**（自然语言创建）/ **Webhook**（外部 HTTP 触发）/ **群记忆**（按群积累）/ **知识收件箱**（问答沉淀规则待主人裁决）。
- 全局唤起热键（默认关，设置里可配可换）。
- 运行时动效：启动闪屏、节点链路流线（只点亮真实路径，多群并发各自点亮）、会话扫光、完成小红旗、桌面完成小人。
- 节点状态药丸（运行中 / 出错）、连线运行轨迹（累计触发次数 + 最近时间）。

### 📦 工程 / 发布
- **绿色部署**：Tauri v2 打包，主程序 exe + 引擎 sidecar exe 两个文件即可运行；单实例防重开、关窗即净退。
- `rebuild-deploy.bat` 一键 构建 → 部署 → 重启。
- 许可证：**GPL-3.0**（强 copyleft）。第三方依赖声明见 `THIRD-PARTY-NOTICES.md`（`scripts/gen-notices.cjs` 自动生成，含 JS + Rust 依赖）。
