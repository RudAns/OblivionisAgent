## [0.4.0] - 2026-06-17

### 🔒 重点修复
- **主人不再被自己的审批卡拦住**：审批门控（`--permission-prompt-tool` + fork-settings 把 Bash/Edit 强制成 `ask`）原先在 `approvalMode=true` 时无条件挂上，会**盖过主人的 `bypassPermissions`**——于是主人自己提问也弹「工具审批」卡，还会**阻塞最多 100s 等审批**（连带「正在输入」一直不消、卡片点了像没反应）。改为仅当 `permissionMode!=="bypassPermissions"` 时才挂门控：主人免审、访客（default 模式）审批流不变。一处修复同时消除主人侧的三个现象。

### 🧩 技能
- **技能节点支持渐进式披露**（仿 Claude 原生 Skills）：技能文件带 `description:` frontmatter → 系统提示只注入「名字 + 用途 + 文件路径」指针（省上下文），模型命中场景时自己用 Read 读完整正文；不带 frontmatter 的技能维持「全文常驻」（向后兼容）。「报告型 HTML 样式」技能即用此机制，正文可写很全也不占日常上下文。

### 🎛️ 画布交互
- **左键框选**（可多选节点一起拖动）+ **中键（按下滚轮）平移** + **按住空格 + 左键平移**（照顾没鼠标中键的情况）；右键保留给上下文菜单。

### 🖥️ 桌面 / 文档
- 状态栏版本号改为**动态读真实版本**（不再写死，发版不会再漏改）。
- 新增软件介绍页 `docs/presentation.html`，报告 / 演示型 HTML 设计规范落在 `docs/html-design.md`（含可贴进技能节点的精简稿）。

## [0.3.0] - 2026-06-17

本期重点是**多会话流水线（L5）**——把多个本地会话串成流水线协作；外加一批可靠性加固，以及若干桌面信息一览（需重建生效）。

### 🔗 编排（本期重点）
- **多会话流水线（L5）**：画布支持「会话 → 会话」连线；一个会话回复后，沿连线把产出喂给下游会话继续加工，逐级把结果发回同群（`⛓` 前缀）。让多个本地会话各司其职、串成一条流水线协作。完全加性 + opt-in（router 对会话是终点，对现有路由零影响）；visited 去重 + 最多 3 跳防环，下游产出出站前脱敏。

### 🔒 可靠性 / 安全
- **cron 看门狗 + 抖动**：单次运行超 10 分钟未结束强制复位（修「卡死后永不再触发」）；同一 tick 多个到点错峰启动，避免同秒齐发撞飞书限频。
- **config 原子写 + 备份**：先写临时文件再 rename（写一半崩了不污染主文件）；落盘前备份 `.bak`，主配置损坏时自动从备份恢复；加 schema 迁移挂钩。
- **webhook HMAC + 限流**：webhook 节点可设 HMAC 密钥，校验 `X-Hub-Signature-256` / `X-Signature` 签名头防伪造回调；每 token 滑窗限流 60 次/分钟。

### 🖥️ 桌面体验
- **节点 / 会话状态更醒目**：会话侧栏圆点（运行 / 出错）加发光 + 脉冲，出错会话卡加红色左条；节点出错态改为缓慢脉冲（`prefers-reduced-motion` 下不动）。
- **文档查看器加「复制路径」**：文件树右键 + 文档头部按钮。
- **禁用 webview 原生右键菜单**（可编辑文本框除外）。

### 📊 可观测性 / 信息一览（读本地缓存 / 配置，不调 API、不耗 token）
- **成本看板（悬浮框）**：一键弹出盖在主界面上的浮框——累计 / 今日花费、按会话柱状、近 14 天迷你柱图、最近运行明细；点别处自动收起。数据来自每次会话运行的 stream-json `cost_usd`，记到 `~/.oblivionis/costs.jsonl`。
- **上下文用量悬停**：状态栏当前终端名旁显示 `%`，悬停看上下文体量（固定开销 / 对话消息 / 空闲）+ 上次回合输出。读 transcript 估算；mtime 门控 + 只读文件头尾窗口，空闲近零开销。
- **顶栏「活动趋势」小标**：chip 本身是近 7 天迷你趋势图；悬停浮框 = 本月日历热力图（含星期 / 日期）+ Claude `/stats` 同款全量统计（常用模型 / 总 token / 会话数 / 最长会话 / 活跃天数 / 连续活跃 / 最活跃日）。读 `~/.claude/stats-cache.json`。
- **左上角品牌悬停**：本软件版本 + 构建时间，以及 Claude CLI 版本 / 登录账号 / 套餐（读 `~/.claude.json`，敏感令牌不碰）。
- 彩蛋：上面活动浮框里附了个「今日宜忌」开发版黄历（85 条程序 + 游戏开发梗，每天稳定随机，图一乐）。

## [0.1.0] - 2026-06-16

首个公开版本。把飞书群聊接入本地 Claude Code 会话的 Windows 桌面工具。

### 🧩 核心
- **飞书群 ↔ 本地 Claude 会话桥接**：群里 @机器人 → 路由到你电脑上对应项目的 claude 会话 → 富文本回帖并 @提问人。
- **订阅合规**：一切 LLM 调用都通过遥控**官方 `claude` CLI** 完成，绝不直连 API / 碰订阅 OAuth 令牌。
- **连线式画布**（React Flow）：飞书群 → 路由 → Claude 会话，拖节点连线即接入；可折叠成会话卡片专注终端。
- **两会话模型**：`base`（你的开发终端会话，永不被飞书污染）+ `fork`（脱敏分身，所有飞书消息都走它）。
- **多群多会话 + 意图分流**：不同群路由到不同项目（各自 cwd / 模型 / 权限）；同群消息按 LLM 语义意图走不同分支。

### 🔗 节点类型
- 飞书群 / 路由 / 意图分流 / Claude 会话 / 定时任务 / Webhook。
- **赋能节点三件套**：🎭 人格（SOUL.md）/ 🧩 技能（SKILL.md）/ 🦾 子代理（Claude Code 原生 subagent，在独立上下文做重活、claude 自动委派）——连到会话的赋能口即生效，一个节点可共享给多个会话。
- 赋能连线用**虚线 + 按类型上色**与消息流区分。
- 「**重锚人格**」：改完人格 / 刚连上会话后一键把当前人格重新注入所连会话（保留对话记忆），比「刷新快照」更轻。

### 🔐 安全
- **访客隔离**：访客走 fork 脱敏分身；fork 时 transcript 抹密钥；回复**出站二次脱敏** + 访客安全护栏。
- **飞书审批卡**：访客的敏感操作（改文件 / 执行命令）弹交互卡到群里，主人 [允许 / 拒绝] 才执行；裁决或超时后卡片自动更新状态、去按钮。
- **App Secret 存 Windows 凭据管理器**：不再明文落 `config.json`、不经 WS 广播；首次启动自动从旧明文迁移。
- 主人 / 访客分级 permission mode；全量入站审计落盘。

### 💬 飞书侧交互
- 自助命令：`/status`（状态卡）· `/doctor`（自检）· `/retry`（重跑）· `/continue`（续跑），仅主人可用。
- 读飞书 **docx / Wiki / Sheets / 多维表** 喂上下文。
- **读消息里的文件附件**：随消息（或被引用消息）发来的文本类文件自动读出正文喂给 claude，二进制给本地路径让其用 Read 打开；外部内容用随机围栏隔离，当资料不当指令。
- **长回复自动作为飞书文件回传**，不撑爆气泡。
- 回复含 Markdown 表格 → 飞书**原生表格**。
- **逐工具进度**流式卡：实时显示「正在 运行命令 / 读取文件…」。

### 🖥️ 桌面体验
- **内置交互终端**：双击节点打开开发会话（完整历史回放）；多终端保活·拖拽排序、剪贴板贴图喂图、字号缩放（设置滑杆 / Ctrl±）。
- **中英双语界面**：设置里一键切换 中文 / English，技术标识符保持原样，漏译自动回退中文。
- **会话视图粘滞**：左侧选会话只换「哪个会话」，终端 / 转录视图用面板顶部小页签切换，不再强制跳终端。
- **定时任务**（自然语言创建）/ **Webhook**（外部 HTTP 触发）/ **群记忆**（按群积累）/ **知识收件箱**（问答沉淀规则待主人裁决）。
- 全局唤起热键（默认关，设置里可配可换）。
- 运行时动效：启动闪屏、节点链路流线（只点亮真实路径，多群并发各自点亮）、会话扫光、完成小红旗、桌面完成小人。
- 节点状态药丸（运行中 / 出错）、连线运行轨迹（累计触发次数 + 最近时间）。

### 📦 工程 / 发布
- **绿色部署**：Tauri v2 打包，主程序 exe + 引擎 sidecar exe 两个文件即可运行；单实例防重开、关窗即净退。
- `rebuild-deploy.bat` 一键 构建 → 部署 → 重启。
- 许可证：**GPL-3.0**（强 copyleft）。第三方依赖声明见 `THIRD-PARTY-NOTICES.md`（`scripts/gen-notices.cjs` 自动生成，含 JS + Rust 依赖）。
