# 技能 · 报告式 HTML 风格（Claude 暖色书卷风）

当被要求生成**报告 / 演示 / 文档型**（非玩法、非游戏化）HTML 时，套用下面这套视觉规范（参考 Anthropic Claude 官网风格）：

## 配色（暖色，禁止冷蓝灰）
- 页面底 羊皮纸 `#f5f4ed`；卡片 象牙 `#faf9f5`；暗段 近黑 `#141413` / 深灰 `#30302e`。
- 品牌强调（仅主 CTA / 关键时刻）赤陶 `#c96442`；暗底链接 / 强调 珊瑚 `#d97757`。
- 正文 `#141413`；次要 橄榄灰 `#5e5d59`；三级 石灰 `#87867f`；暗底正文 暖银 `#b0aea5`。
- 边框 奶油 `#f0eee6` / 暖砂 `#e8e6dc`（暗底 `#30302e`）。任何灰都带黄褐暖调。

## 字体
- 标题用衬线（`Newsreader`，回退 `Georgia`），**统一 weight 500**（不加粗 >500），行高紧 1.10–1.30。
- 正文 / UI 用无衬线（`Inter` / `system-ui`），正文行高松 **1.60**。
- 代码用等宽，且仅限代码。标题禁用无衬线、正文禁用等宽。

## 组件
- 卡片 象牙底 + `1px solid #f0eee6` + 圆角 16px（强调容器 16–32px）。
- 阴影优先**环形** `0 0 0 1px` 暖灰，或极轻投影 `rgba(0,0,0,0.05) 0 4px 24px`；不用重投影。
- 按钮圆角 8–12px：主 CTA 赤陶底象牙字；次按钮 暖砂 `#e8e6dc` 底 炭灰 `#4d4c48` 字。截图 / 媒体圆角 16–32px。

## 布局节奏
- 容器最大 ~1120px 居中；主分区上下留白 80–120px。明 / 暗段交替制造「章节感」。

## 禁忌
冷蓝灰、纯白页底、饱和杂色、尖角（<6px）、重投影、衬线加粗、标题用无衬线、正文行高 <1.4、几何科技风插画。

> 仅对「报告 / 演示 / 文档型」HTML 生效；玩法 / 游戏化 / 交互玩具型 HTML 不受此约束。
