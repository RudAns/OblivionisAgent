# 截图放这里

把下面这几张 PNG 放进本目录，根目录 README 的「📸 界面预览」就会自动显示（文件名要对得上）：

| 文件名 | 拍什么 |
|---|---|
| `canvas.png` | 连线配置画布：飞书群 → 路由 → Claude 会话 的整条链路 |
| `terminal.png` | 内置交互终端：开着 claude 会话、最好带一张贴图缩略图或运行时扫光 |
| `intent.png` | 意图分流：一个群分到两条带「意图条件」的边 |
| `feishu.png` | 飞书里 @机器人 后机器人富文本回帖、@提问人的效果 |

建议：

- 宽度 1200–1600px、深色主题，截清楚就行；GitHub 会自动按表格列宽缩放。
- 想多放几张可在 README 的表格里继续加行/列，路径都用 `docs/screenshots/xxx.png`。
- 体积大的话用 PNG 压一下（TinyPNG 之类），别让仓库太重。
