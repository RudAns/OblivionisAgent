连线式画布

🔐

工具审批卡

🎭

赋能节点

人格（SOUL.md）/ 技能（SKILL.md）/ 子代理都做成可连线节点，连到会话即生效，一格可共享多会话。

为什么这样设计

遥控本地 CLI，而不是调 API

很多人用的是 Claude 订阅（Pro / Max）而非 API Key。本项目一切 LLM 调用都通过驱动官方 claude CLI 完成。

✓ 订阅合规

Anthropic 禁止第三方工具直接使用订阅 OAuth 令牌；本项目绝不碰令牌、只遥控官方 CLI——合规、稳妥。

✓ 零额外成本

复用你已有的订阅登录态，不产生额外 API 账单；也完整复用已有的会话历史与项目上下文。

✓ 数据留在本地

会话、配置、审计全部落在你自己的电脑；飞书 App Secret 存进 Windows 凭据管理器，不明文落盘。

✓ 访客始终受控

访客走脱敏分身、回复二次脱敏；敏感操作要主人在飞书审批卡上点「允许」才执行。
